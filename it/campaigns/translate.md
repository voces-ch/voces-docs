# Tradurre una campagna

voces.ch rende semplice creare campagne multilingue. Quando incorpori il widget, usa il parametro `lang` (es: `lang: 'it'`) per scegliere la lingua mostrata.

## Gestire le traduzioni

1. Apri la campagna.
2. Clicca **Manage translations**.
3. Compila i campi per le lingue attive.
4. Salva.

## Campi traducibili

- **Title\***
- **Slug\***
- **Description**
- **Submit Label\***
- **Success Message** o **Success URL**
- **Public Label\*** dei campi personalizzati
