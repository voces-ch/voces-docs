# Crea una campagna

Dopo aver registrato la tua organizzazione, puoi creare la tua prima campagna (ad esempio una petizione o una lettera aperta).

## Come iniziare

1. Nel menu principale a sinistra clicca **Campagne**.
2. In alto a destra clicca **Crea**.

Ora sei nell'editor campagna.

## 1. Informazioni base

- **Attiva (Is Active):** metti in pausa o attiva una campagna in qualsiasi momento.
- **Titolo:** nome pubblico della campagna.
- **Testo pulsante (Submit Label):** testo del pulsante invio.
- **Obiettivo firme:** se imposti un numero (es. `1000`), puoi mostrare una barra di progresso.

## Data pooling (funzione avanzata)

::: danger Attenzione: solo per utenti esperti
Questa funzione influisce su visibilita dati e responsabilita legale. Attivala solo se conosci bene le implicazioni.
:::

Per default i dati sono separati. Con **data pooling**, l'host puo vedere anche le firme raccolte dai partner. I partner continuano a vedere solo i propri dati.

## 2. Azione di successo

Dopo l'invio puoi:

- **Mostrare un messaggio di successo**
- **Reindirizzare a un URL** (completo, con `https://`)
