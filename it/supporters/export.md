# Esportare i sostenitori

I dati campagna sono fondamentali. Con voces.ch mantieni il pieno controllo e puoi esportare i sostenitori (CSV o Excel) per CRM o newsletter.

## 1. Export singola campagna

1. Apri la campagna.
2. Clicca **Export**.
3. Ricevi un file con le firme di quella campagna.

## 2. Export globale (tutte le campagne)

1. Menu **Data** -> **Signatures**.
2. Seleziona le voci desiderate.
3. Avvia l'export.

## Note sui campi personalizzati

- I campi con la stessa **internal key** vengono uniti in una sola colonna.
- Celle vuote sono normali se un campo non esiste in tutte le campagne.
