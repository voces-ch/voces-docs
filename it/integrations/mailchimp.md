# Integrazione Mailchimp

Con l'integrazione Mailchimp, le persone che firmano la tua campagna vengono sincronizzate automaticamente nella tua audience email.

## Configurazione

1. Apri la campagna nel backend voces.ch.
2. Vai su **Integrations** e scegli **Mailchimp**.
3. Attiva l'integrazione e compila i campi.

### 1. Credenziali API

- **API Key**
- **Audience ID (List ID)**

### 2. Controllo opt-in

In **Opt-In Checkbox** scegli il campo consenso newsletter.

### 3. Mappatura campi (merge fields)

Mappa i campi voces ai tag Mailchimp.

Esempio:

- `email` -> `EMAIL`
- `first_name` -> `FNAME`
- `last_name` -> `LNAME`
