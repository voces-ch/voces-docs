# Pages de campagne hebergees

Pas de site web pour votre campagne ? Utilisez les **hosted pages voces.ch**.

## 1. Creer une page

Dans le dashboard:

1. Cliquez sur **Campaign Pages**.
2. Cliquez sur **New Campaign Page**.

## 2. Parametres de page

- **Published**: rend la page publique.
- **Campaign**: campagne associee.
- **Locale**: langue de la page.
- **URL Slug**: lien court.
- **Theme**: style visuel.

## 3. Contenu

Dans **Page content**, redigez votre histoire de campagne.

## 4. Publier et partager

Activez **Published**, enregistrez, puis partagez votre page:

`https://act.voces.ch/votre-slug`
