# Objectif de campagne et barre de progression

Un objectif visible motive les soutiens et rend les progres transparents.

## Activer l'objectif

Dans la campagne, onglet **Campaign Goal**, activez l'objectif.

## Type d'objectif

### 1. Compter les soumissions

Chaque soumission valide compte pour **1** (avec deduplication entre partenaires).

### 2. Additionner la valeur d'un champ

Additionne les valeurs d'un champ numerique, utile pour promesses de dons ou autres objectifs quantifies.

## Valeur cible

Dans **Goal Target**, saisissez la valeur cible (ex: `10000`).
