# Creer une campagne

Une fois votre organisation enregistree, vous pouvez creer votre premiere campagne (par exemple une petition ou une lettre ouverte).

## Demarrage

1. Dans le menu de gauche, cliquez sur **Campagnes**.
2. En haut a droite, cliquez sur **Creer**.

Vous etes maintenant dans l'editeur de campagne.

## 1. Informations de base

- **Actif (Is Active):** Mettre en pause ou activer la campagne a tout moment.
- **Titre:** Nom public de la campagne.
- **Texte du bouton (Submit Label):** Texte affiche sur le bouton d'envoi.
- **Objectif de signatures:** Si vous indiquez un nombre (par exemple `1000`), vous pouvez afficher une barre de progression.

## Data pooling (fonction avancee)

::: danger Attention: reserve aux utilisatrices et utilisateurs experimentes
Cette fonctionnalite modifie la visibilite des donnees et la responsabilite legale. Activez-la uniquement si vous comprenez pleinement ses implications.
:::

Par defaut, les donnees sont separees. Avec le **data pooling**, l'organisation host peut lire toutes les signatures, y compris celles collectees par les partenaires. Les partenaires continuent de voir uniquement leurs propres donnees.

## 2. Action de succes

Apres soumission du formulaire, vous pouvez:

- **Afficher un message de succes**
- **Rediriger vers une URL** (complete, incluant `https://`)
