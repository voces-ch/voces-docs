# Traduire une campagne

voces.ch facilite les campagnes multilingues. Lors de l'integration du widget, utilisez le parametre `lang` (ex: `lang: 'fr'`) pour choisir la langue affichee.

## Gerer les traductions

1. Ouvrez la campagne.
2. Cliquez sur **Manage translations**.
3. Renseignez les champs pour les langues activees.
4. Enregistrez.

## Champs traduisibles

- **Title\***
- **Slug\***
- **Description**
- **Submit Label\***
- **Success Message** ou **Success URL**
- **Public Label\*** des champs personnalises
