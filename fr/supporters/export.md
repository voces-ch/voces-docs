# Exporter les soutiens

Les donnees de campagne sont essentielles. Avec voces.ch, vous gardez le controle total et pouvez exporter vos soutiens (CSV, Excel) pour vos outils CRM ou newsletter.

## 1. Export d'une seule campagne

1. Ouvrez la campagne.
2. Cliquez sur **Exporter**.
3. Vous recevez un fichier avec les signatures de cette campagne.

## 2. Export global (toutes les campagnes)

1. Menu **Data** -> **Signatures**.
2. Selectionnez les entrees voulues.
3. Lancez l'export.

## Particularites des champs personnalises

- Les champs avec la meme **internal key** sont fusionnes en une seule colonne.
- Des cellules vides sont normales si un champ n'existe pas dans toutes les campagnes.
