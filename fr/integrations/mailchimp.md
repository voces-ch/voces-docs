# Integration Mailchimp

Avec l'integration Mailchimp, les personnes qui signent votre campagne sont synchronisees automatiquement vers votre audience email.

## Configuration

1. Ouvrez votre campagne.
2. Allez a **Integrations** et choisissez **Mailchimp**.
3. Activez l'integration et renseignez les champs.

### 1. Identifiants API

- **API Key**
- **Audience ID (List ID)**

### 2. Verification opt-in

Choisissez dans **Opt-In Checkbox** le champ de consentement newsletter.

### 3. Correspondance des champs (merge fields)

Mappez les champs voces vers les tags Mailchimp.

Exemple:

- `email` -> `EMAIL`
- `first_name` -> `FNAME`
- `last_name` -> `LNAME`
