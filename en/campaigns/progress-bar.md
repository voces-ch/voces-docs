# Campaign goal and progress bar

A visible goal motivates supporters and transparently shows campaign momentum. In voces.ch, you can define a goal for each campaign and display it in the widget as a progress bar.

## Enable campaign goal

In the backend, open your campaign and go to the **Campaign Goal** tab.

Turn on **Enable Campaign Goal**.

## Define goal type

voces.ch offers two ways to measure progress.

### 1. Count submissions

Default mode for classic petitions. Each successful form submission counts as **1** (with duplicate handling across partner organizations).

- Example: You want 10,000 signatures. Each signer moves the progress bar forward.

### 2. Sum a field value

Advanced mode that sums values from a numeric form field. Useful for crowdfunding, donation pledges, or collecting hours/kilometers.

- Requirement: At least one **Number** field must exist in campaign fields.
- Example: Ask "How many trees do you want to plant?" If person A enters 5 and person B enters 10, progress becomes 15.

When this mode is selected, the **Goal Field** dropdown appears. Choose the numeric field whose values should be summed.

## Define goal target

In **Goal Target**, enter the absolute value you want to reach.

Use plain numbers only, without separators. For example enter `10000`. This is the 100% value for the progress bar.
