# Hosted campaign pages

No website yet for your project, or you want to start collecting signatures quickly? Use **voces.ch hosted pages**.

You can create a professional standalone campaign landing page in minutes and publish it under the short `act.voces.ch` domain.

## 1. Create a new page

In your voces.ch dashboard:

1. Click **Campaign Pages** in the left menu.
2. Click **New Campaign Page** in the top-right.

## 2. Configure page settings

In **Page details**, define basic page properties:

- **Published (Live):** If enabled, page is publicly accessible.
- **Campaign:** Select which campaign this page belongs to. The campaign widget will be embedded automatically at the bottom.
- **Locale:** Select page language. Only languages enabled in campaign settings are shown.
- **URL Slug:** Page URL slug. voces.ch suggests one based on campaign title.
- **Theme:** Choose visual style, for example "Default" or "Impact".

## 3. Build content

In **Page content**, use the editor to tell your campaign story. This content appears above the signature form.

Editor basics:

- Click and start typing.
- Type `/` to open the block menu.
- Insert headings, lists, quotes, or images.
- Select text to format bold, italic, or links.

## 4. Publish and share

When ready:

1. Turn on **Published**.
2. Save the page.

Your page is now live.

URL format:

`https://act.voces.ch/your-slug`

The signature widget (including configured fields and privacy checkboxes) is inserted automatically at the end of your content.
