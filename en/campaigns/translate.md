# Translate a campaign

voces.ch makes it easy to run multilingual campaigns and reach broader audiences. When embedding the widget, use the `lang` parameter (for example `lang: 'fr'`) to control which language visitors see.

## Manage translations

Translation is built directly into the backend:

1. Open your campaign (detail or edit view).
2. Click **Manage translations**.
3. You are redirected to the edit page where you can fill campaign content for all enabled languages.
4. Enter translated texts in the relevant fields and save.

## Translatable fields

Not every field needs translation. Internal names or numeric goals are typically language-independent. The following fields can be translated per language.

Required fields are marked with `*`.

### Base information

- **Title***: Public campaign name.
- **Slug***: URL-friendly slug (for example _save-the-lake_).
- **Description**: Campaign background text.
- **Submit Label***: Button text (default is language-aware but can be overridden).

### Success action

- **Success Message**: Thank-you text shown after signing (if that option is selected).
- **Success URL**: Redirect URL (if that option is selected).

Either success message or success URL is required.

### Custom campaign fields

- **Public Label***: Label shown above each field (for example _First name_ instead of _Vorname_).
